PaywallBundle
=============

This bundle provides easy integration of the Superdesk Publisher Paywall component with any Symfony full-stack application.

Superdesk Publisher uses this bundle for the paywall implementation.

Documentation
-------------

Documentation is available here: [http://superdesk-publisher.readthedocs.org](http://superdesk-publisher.readthedocs.org/en/latest/bundles/SWPPaywallBundle/index.html)

How to Contribute
-------------

You want to contribute to this bundle? That's great and we really appreciate it! 
Before sumbitting your work though, please read the contributing 
[guideline](http://superdesk-publisher.readthedocs.org/en/latest/contributing/index.html) 
for the details on how to do it. Thanks!
